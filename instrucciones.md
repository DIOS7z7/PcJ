# 🎮 INSTRUCCIONES DE USO
 ## 1. CREAR LA MÁQUINA
  1. Ejecutá el archivo `crear-maquina.bat` (solo si estás en una computadora con VirtualBox)
   2. Esperá a que termine de correr todo el código
    3. Se creará una máquina llamada `Steam-Gaming-PC`
     ## 2. INSTALAR WINDOWS 11
      1. Abrí VirtualBox (en tu computadora)
       2. Hacé doble clic en: `Steam-Gaming-PC`
        3. Seleccioná el archivo de instalación de Windows 11 (.iso)
         4. Seguí los pasos: todo es automático
          ## 3. CONFIGURAR PUERTO
           El puerto 3389 ya está abierto y listo para usar. Sirve para entrar a la máquina desde cualquier dispositivo.
            ## 4. PREPARAR PARA JUGAR
             1. Instalá las extensiones de VirtualBox
              2. Bajá e instalá Steam
               3. Instalá los controladores de gráficos
                4. En Steam: Configuración > Descargas > Activar "Descargas rápidas"
                 ## 💡 CONSEJOS
                  ✅ No abrás otros programas mientras jugás
                   ✅ Bajá la calidad de gráficos si van lentos
                    ✅ Usá la configuración de `configuracion.txt`
